# GeoFS Aircraft Duplicator
Duplicates a GeoFS Aircraft Model in-game
- Use H to hide the freeze button
- Only one livery can be applied per aircraft
- When you click the button you must switch to a different aircraft to refresh the physics model. You may switch back into the previous aircraft once you do that.
## How to install

Click on the file that says "Userscript.js"

Copy the contents of it, and paste it into your script loader.

Additionally you can also paste it into your browser console
